🚀 Deployment Information

📋 Pre-Deployment Steps:

  1. Firebase Setup: Update `js/firebase-config.js` with your Firebase project credentials
  2. Telegram Bot: Update bot token and chat ID in `confession.html`
  3. Test Locally: Serve with a local web server (e.g., `python -m http.server 8000`)
  4. Deploy to Vercel: Run `vercel` command or connect GitHub repo
